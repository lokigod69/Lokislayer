## üé≤ THE ENTRY EXPERIENCE

### First Load Sequence
1. **Brief boot/loading sequence** (glitchy, atmospheric, 2-3 seconds max)
2. **Liminal Lobby appears** with three options:
   - üé≤ **Roll the Dice** ‚Äî Random interface
   - üîò **Choose Your Reality** ‚Äî See all 6 thumbnails
   - üß† **Enter Neural Map** ‚Äî Go directly to default

### The Dice Mechanic
- Visual 3D six-sided dice
- User clicks to roll
- Satisfying physics animation (tumble, bounce, land)
- Each face mapped to an interface (1-6)
- Transition animation (glitch/warp/fade) to selected interface
- **Return option**: "Roll Again" button always accessible in corner

### Manual Selection
- Grid or carousel of all 6 interfaces
- Each has thumbnail preview and name
- Click to enter that experience
- Can also access from any interface via menu
